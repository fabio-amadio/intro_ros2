# intro_ros2

Lab material for the **Introduction to ROS 2** course.

## Getting the Docker image

A pre-built Docker image is available for this tutorial.

Pull the image with:

```bash
docker pull ghcr.io/fabio-amadio/intro-ros2:2026
```

After the image has been downloaded, start the container and open a terminal inside it by running:

```bash
bash run_ros2.sh
```
